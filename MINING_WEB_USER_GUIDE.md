## 📱 Quick Start

### 1. Install MetaMask
If you don't have a Web3 wallet:
- Visit https://metamask.io
- Install browser extension
- Create new wallet
- Save seed phrase securely

### 2. Connect Wallet
1. Visit https://mining.reagent.eu.cc
2. Click **"Connect Wallet"** button
3. Approve connection in MetaMask
4. Done! ✅

### 3. Add Tempo Network
The interface will automatically:
- Detect your current network
- Prompt to switch to Tempo
- Add Tempo Network if needed
- No manual configuration required!

### 4. Start Minting
1. Enter amount (1-10,000 REAGENT)
2. Click **"Estimate Gas"**
3. Review fees
4. Click **"Mint"**
5. Confirm in wallet
6. Wait for confirmation

## ⚠️ Common Issues

### "Please install MetaMask"
**Problem:** No Web3 wallet detected  
**Solution:** Install MetaMask or compatible wallet

### "Failed to switch to Tempo Network"
**Problem:** User rejected network switch  
**Solution:** 
1. Open MetaMask
2. Click network dropdown
3. Select "Tempo Network"
4. Or click "Add Network" and retry

### "Insufficient funds"
**Problem:** Not enough PATH for gas  
**Solution:** 
1. Get PATH tokens
2. Send to your wallet
3. Retry minting

### "Please connect your wallet first"
**Problem:** Wallet not connected  
**Solution:** Click "Connect Wallet" button

### "Please enter a valid amount"
**Problem:** Invalid input  
**Solution:** Enter amount between 1-10,000
